1️⃣ What is the difference between var, let, and const?

Answer:
var is the old way to create variables. It is better to avoid it now.
let is the modern way to create a variable when you need to change its value later.const is the modern way to create a variable when the value will never change.

2️⃣ What is the spread operator (...)?

Answer:
spread operator is used to spread the elements of an array or an object into individual elements.It is useful for copying arrays or objects, combining them, or passing them as arguments to functions.

3️⃣ What is the difference between map(), filter(), and forEach()?

Answer:
forEach(): Just loops through each item to do something. It does not return a new array back.
map(): Changes every item and returns a brand-new array with those changed items.
filter(): Checks a condition and returns a brand-new array containing only the items that passed the condition.

4️⃣ What is an arrow function?

Answer:
It is a shorter and cleaner way to write a function in JavaScript using the => symbol.It was introduced in ES6.

5️⃣ What are template literals?
Answer:
It is a string written inside backticks (`). It lets you easily write text on multiple lines and put variables directly inside the text using ${variableName}.
